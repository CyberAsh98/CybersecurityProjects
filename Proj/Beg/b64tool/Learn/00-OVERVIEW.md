# Overview

## What This Project Teaches

This project bridges:

- Security engineering
- Encoding theory
- Malware analysis fundamentals
- CLI design patterns
- Confidence-based detection algorithms

You will learn how encoding works at the byte level and how attackers abuse it.

---

## Prerequisites

- Python 3.12+
- Basic command line familiarity
- Understanding of bytes vs strings (helpful but not required)

---

## Real-World Applications

- JWT token inspection
- API debugging
- WAF bypass testing
- Malware configuration decoding
- Data exfiltration detection
- CTF challenges
